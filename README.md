# 🚀 TaskNest

A modern and responsive **Task Management Application** built with **React.js** that helps users organize daily tasks efficiently. TaskNest allows users to create, edit, complete, delete, and manage tasks with an intuitive interface while automatically saving data using Local Storage.

---

## ✨ Features

- ➕ Add new tasks
- ✏️ Edit existing tasks
- ✅ Mark tasks as Complete / Undo
- 🗑️ Delete individual tasks
- 🧹 Clear all tasks with confirmation
- 📊 Progress Tracker
- 🏷️ Task Priority (High, Medium, Low)
- 📂 Task Categories (Work, Personal, General)
- 💾 Persistent Local Storage
- 📱 Responsive Design
- 🎨 Modern Glassmorphism UI
- ⚡ Smooth hover animations

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/harsha70900/tasknest.git
```

### Navigate to the Project

```bash
cd tasknest
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

---

## 💡 How It Works

1. Create a new task.
2. Select a priority and category.
3. Mark tasks as completed or undo them.
4. Edit tasks anytime.
5. Delete unwanted tasks.
6. Track your completion progress.
7. All tasks are automatically saved in Local Storage.
